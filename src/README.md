# Projeto Média de Aluno

Este projeto calcula a média de um aluno com base em três notas.

## Funcionalidades
- Leitura de 3 notas
- Cálculo da média
- Exibição da situação do aluno (Aprovado, Recuperação ou Reprovado)

## Tecnologias utilizadas
- Java
- Scanner

## Como executar
1. Compile:
   javac MediaAluno.java

2. Execute:
   java MediaAluno
