# media-aluno-java

Este projeto calcula a média de um aluno com base em três notas informadas pelo usuário.

## Funcionalidades
- Leitura de 3 notas
- Cálculo da média
- Exibição da situação do aluno (Aprovado, Recuperação ou Reprovado)

## Tecnologias utilizadas
- Java
- Scanner (entrada de dados)
- Console
